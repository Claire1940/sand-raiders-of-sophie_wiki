<Section icon={Trophy} title="Region Locks and Character Progression">

When connecting to the servers, your progression data is handled differently depending on the element. To maintain competitive integrity and reduce latency-induced combat advantages, the game utilizes regional servers. 

Your inventory, accumulated resources, and tech tree progress are strictly bound to the physical region you select upon logging in. However, your hard-earned Trampler designs are saved globally, allowing you to recreate your favorite mobile bases no matter which server you connect to.

| Progression Element | Server Sync Type | Transferable Across Regions? | Impact of Regional Outages |
|---|---|---|---|
| **Character Inventory** | Region-Locked | No | Inaccessible if the specific regional server is offline |
| **Tech Tree Upgrades** | Region-Locked | No | Progress is paused; cannot unlock nodes on other regions |
| **Trampler Blueprints** | Account-Bound | **Yes** | Fully safe; blueprints can be deployed on any server region |
| **Event Rewards** | Account-Bound | **Yes** | Skins (like the Server Slam Einzel skin) remain universally available |

<Callout type="warning" title="Switching Regions">
If you experience high latency on your current server and decide to switch regions, you will be prompted to create a new character. Your items and tech tree progress from your original region will remain safe but will not carry over to the new server.
</Callout>

<Section icon={Shield} title="Troubleshooting Connection and Anti-Cheat Errors">

If the official status channels indicate that the servers are fully operational but you are still unable to connect, the issue is likely client-side. The PC version of the game utilizes BattlEye, a kernel-level anti-cheat system. If BattlEye fails to initialize, or if you are running the game on a non-standard operating system without the proper runtimes, the game servers will automatically reject your connection attempt.

Follow these steps to resolve the most common connection and anti-cheat issues on PC and Linux-based systems.

<Steps>
  <Step num="1" title="Verify Game Files on Steam">
    Right-click the game in your Steam library, select Properties, navigate to the Installed Files tab, and click Verify Integrity of Game Files. This repairs corrupted BattlEye files.
  </Step>
  <Step num="2" title="Configure Proton BattlEye Runtime (Linux/Steam Deck)">
    If you are playing on Linux or a Steam Deck, you must install the Proton BattlEye Runtime from your Steam library. Without this runtime, the anti-cheat will fail to establish a secure handshake with the game servers.
  </Step>
  <Step num="3" title="Clear Your DNS Cache">
    Open your command prompt as an administrator and run the command 'ipconfig /flushdns'. This clears outdated IP routing paths to the matchmaking servers.
  </Step>
  <Step num="4" title="Check Firewall Permissions">
    Ensure that both the game executable and the BattlEye service are whitelisted in your Windows Defender or third-party firewall settings to allow unrestricted incoming and outbound network traffic.
  </Step>
</Steps>

| Operating System | Anti-Cheat System | Required Runtime | Common Error Code |
|---|---|---|---|
| **Windows 10 / 11** | BattlEye Kernel-Level | Standard Windows Service | BattlEye Service initialization failed |
| **Linux / Steam Deck** | BattlEye via Proton | Proton BattlEye Runtime | Disconnected by Server: Authentication Timeout |

<Callout type="tip" title="Uninstallation Warning">
If you decide to uninstall the game, be aware that the BattlEye anti-cheat service may remain on your system. To completely clean your directory, you may need to manually navigate to your common files folder and execute the BattlEye uninstaller.
</Callout>

</Section>
