<Section icon={Shield} title="Server Stability and Technical Challenges">

Despite the successes of the **sand raiders of sophie server slam**, technical hurdles remain a primary concern for the community. The game has faced persistent DDoS attacks, prompting the implementation of new protection layers. A particularly severe issue involves the "VOYAGE" mode, which currently suffers from a memory crash. This crash leaves players stuck on a reconnection screen, effectively locking them out of their current session.

| Issue Type | Symptom | Developer Response |
| :--- | :--- | :--- |
| **Memory Crash** | Disconnects in VOYAGE mode | Avoid VOYAGE until fix |
| **DDoS Attacks** | Server-wide latency and outages | Protections implemented |
| **Extraction Loop** | Loss of loot after successful extraction | Under investigation |
| **Solo Matchmaking** | Solo players matched against large crews | High priority fix |

<Callout type="warning" title="Critical Warning">
If you encounter a "Connection to server lost" message during extraction, do not immediately exit. Attempt to reconnect once, but be aware that persistent loops may indicate a dead server instance due to the memory leak.
</Callout>
