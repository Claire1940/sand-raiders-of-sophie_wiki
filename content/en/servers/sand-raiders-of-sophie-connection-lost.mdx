<Section icon={Target} title="Platform-Specific Fixes for Linux and Steam Deck">

Playing on Linux or Steam Deck requires additional compatibility layers to communicate with the game's anti-cheat systems. If these systems are not configured correctly, the server will reject your connection, resulting in a disconnect. The game utilizes BattlEye anti-cheat, which must be manually configured on Proton-based operating systems.

If you are playing on Linux or Steam Deck, you must install the **Proton BattlEye Runtime** directly from your Steam library to prevent automatic kicks from multiplayer servers.

<Steps>
  <Step num="1" title="Search Steam Library">
    Open your Steam client, head to the Library tab, and search for **Proton BattlEye Runtime** in the search bar.
  </Step>
  <Step num="2" title="Install the Runtime">
    If it is not already installed, click **Install** and allow the download to complete.
  </Step>
  <Step num="3" title="Restart Steam">
    Fully close and restart the Steam client to ensure the runtime integrates properly with the game launcher.
  </Step>
</Steps>
