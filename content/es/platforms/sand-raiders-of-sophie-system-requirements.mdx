<Section icon={Shield} title="Optimización de tu configuración">

Más allá del hardware puro, la configuración de tu entorno de software juega un papel fundamental en el rendimiento del juego. Dado que el juego utiliza el sistema antitrampas a nivel de kernel BattlEye, asegúrate de que tu sistema esté libre de procesos en segundo plano conflictivos.

| Tarea de optimización | Beneficio |
| :--- | :--- |
| **Actualizar controladores de GPU** | Evita artefactos visuales y mejora el ritmo de fotogramas |
| **Instalación en SSD** | Reduce los tiempos de carga del mundo generado procedimentalmente |
| **Limpieza de aplicaciones en segundo plano** | Libera RAM para el complejo motor de físicas de los Tramplers |
| **Comprobación de DirectX 11** | Garantiza la compatibilidad con la API de renderizado principal del juego |

<Callout type="warning" title="Información del sistema antitrampas">
Este juego utiliza BattlEye, un sistema antitrampas a nivel de kernel. Si decides desinstalar el juego, recuerda que es posible que debas eliminar manualmente los archivos de BattlEye para borrarlos por completo de tu sistema.
</Callout>

</Section>
