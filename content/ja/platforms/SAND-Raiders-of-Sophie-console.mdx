<Section icon={Swords} title="ゲームモード：ボヤージュ vs ストームダイブ">

**SAND: Raiders of Sophie コンソール版**で生き残るためには、ゲームモードの違いを理解することが不可欠です。各モードでリスクとリターンの比率が異なり、トランプラーのビルド構成にも影響を与えます。

| 特徴 | ボヤージュモード | ストームダイブモード |
| :--- | :--- | :--- |
| **リスク** | 低リスク | 高リスク |
| **タイマー** | 制限時間なし | 砂嵐による制限あり |
| **戦利品のティア** | 標準 / コモン | レア / 高価値 |
| **PvPの激しさ** | 穏やか | 激しい（バトルロイヤル形式） |
| **最適な用途** | 練習 / 資源稼ぎ | エンドゲーム / 競争プレイ |

<Callout type="info" title="砂嵐のメカニクス">
ストームダイブモードでは、砂嵐が徐々にマップを飲み込み、生き残っているすべてのプレイヤーを中央へと押し流します。これにより強制的に戦闘が発生し、マッチが進むにつれて脱出が非常に困難になります。
</Callout>

### 脱出のサイクル

どのモードであっても、基本的なゲームサイクルは共通しています：
1.  **物資回収（Scavenge）**: 廃墟となった都市、難破船、放棄された通りを探索します。
2.  **略奪（Loot）**: トランプラーの保管庫を資源やアーティファクトで満たします。
3.  **生存（Survive）**: PvEの脅威（アンデッドや自動防衛システム）および敵対プレイヤーから身を守ります。
4.  **脱出（Extract）**: 脱出地点に到達し、戦利品を持って軌道ステーションに帰還します。

</Section>

<Section icon={Shield} title="システム要件とアンチチート">

**SAND: Raiders of Sophie**のコンソール版またはPC版をスムーズに動作させるには、特定の動作環境を満たす必要があります。PC版では11GBのストレージ空き容量が必要で、ソフィーのプロシージャル生成された地形を読み込むためにSSD（ソリッドステートドライブ）の使用が強く推奨されます。

| コンポーネント | 必要動作環境 | 推奨動作環境 |
| :--- | :--- | :--- |

| **プロセッサー** | Intel i5 (第8世代) / Ryzen 5 (2000シリーズ) | Intel i5 (第13世代) / Ryzen 5 (7000シリーズ) |
| **メモリ** | 16 GB RAM | 32 GB RAM |
| **グラフィック** | GTX 1650 / RX 570 | RTX 3080 / RX 6800 XT |
| **DirectX** | バージョン 11 | バージョン 11 |

<Callout type="warning" title="BattlEyeアンチチート">
PC版では、カーネルレベルのアンチチートシステム「BattlEye」が使用されています。Steam DeckおよびLinuxユーザーは、ゲームを起動するためにSteamライブラリから「Proton BattlEye Runtime」をインストールする必要があります。なお、ゲームをアンインストールした際、BattlEyeのファイルを手動でクリーンアップする必要がある場合があります。
</Callout>
