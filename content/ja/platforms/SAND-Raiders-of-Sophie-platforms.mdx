<Section icon={Cpu} title="PCシステム要件と技術仕様">

惑星ソフィーの砂丘を越えて巨大な歩行要塞を操縦するには、ハードウェアが特定のベンチマークを満たしている必要があります。このゲームは、トランブラーの手続き型生成と物理ベースのインタラクションに依存しているため、スムーズな体験にはCPUパフォーマンスとRAM容量が重要になります。

| 仕様 | 最小要件 | 推奨スペック |
| :--- | :--- | :--- |
| **OS** | Windows 10 (64-bit) | Windows 11 (64-bit) |
| **プロセッサー** | Intel i5 (第8世代) / Ryzen 5 (2000) | Intel i5 (第13世代) / Ryzen 5 (7000) |
| **メモリ** | 16 GB RAM | 32 GB RAM |
| **グラフィック** | GTX 1650 / RX 570 / Arc A380 | RTX 3080 / RX 6800 XT |
| **DirectX** | バージョン 11 | バージョン 11 |
| **ストレージ** | 11 GB の空き容量 | 11 GB (SSD推奨) |

ゲームはさまざまなハードウェア構成に合わせてスケールするように設計されていますが、トランブラーの搭乗アクションや砂嵐のエフェクトに伴う重い物理計算は、推奨ティアで処理するのが最適です。

<CardGrid cols={2}>
  <Card title="ストレージとパフォーマンス">
    - **SSD優先**: 容量は小さい（11 GB）ですが、SSDを使用することで軌道ステーションと惑星表面の間のロード時間が大幅に短縮されます。
    - **VRAM使用量**: 廃墟都市の高解像度テクスチャには、安定したフレームレートを維持するために少なくとも4 GBのVRAMが必要です。
  </Card>
  <Card title="ネットワークと接続性">
    - **ブロードバンド必須**: 永続的なPvPvE体験であるため、安定したインターネット接続が不可欠です。
    - **Ping感度**: 遅延が高いと、長距離砲戦中のトランブラーの操舵や弾丸の着弾判定に影響を与える可能性があります。
  </Card>
</CardGrid>

| クルーの人数 | サーバー容量 | マッチメイキングのロジック |
| :--- | :--- | :--- |
| **ソロプレイヤー** | 最大15人 | ソロのスカベンジャーを他のソロとマッチング |
| **小規模クルー (2-3人)** | 最大36人 | 機敏なレイダーチーム向けに設計 |
| **大規模クルー (4-6人)** | 最大45人 | 歩行要塞による高強度の戦闘 |

<Section icon={PlayCircle} title="はじめに：最初のセッションのセットアップ">

惑星ソフィーへの移行には、単なる購入以上の準備が必要です。メカニカルベースを準備し、さまざまな **SAND Raiders of Sophie プラットフォーム** の運用ルールを理解する必要があります。

<Steps>
  <Step num="1" title="公式な入手">
    [Steamストア](https://store.steampowered.com/app/1431300/SAND_Raiders_of_Sophie/)でゲームを購入するか、コンソールのウィッシュリストに追加してください。「サーバースラム」スキンなどの正当なローンチ報酬が含まれていないことが多いため、サードパーティのキー再販サイトは避けてください。
  </Step>
  <Step num="2" title="トランブラー・エディター">
    最初の出撃前に、トランブラー・エディターに入ります。エンジン、リアクター、ストレージを備えた基本的なレイアウトを構築します。これを設計図として保存してください。ソフィーでトランブラーが破壊された場合でも、設計図があれば軌道ステーションですぐに再構築できます。
  </Step>
  <Step num="3" title="モード選択">
    「ヴォヤージュ（Voyage）」と「ストームダイブ（Storm Dive）」のどちらかを選択します。ヴォヤージュは永続的でリスクの低い探索モードで、マップを学ぶのに理想的です。ストームダイブはハイリスク・ハイリターンなモードで、砂嵐がプレイヤーを中心部へと追い込み、激しいPvPのボトルネックを生み出します。
  </Step>
  <Step num="4" title="エクストラクション・ループ">
    マップに入り、都市の廃墟や難破船を略奪し、アイテムをトランブラーの内部コンパートメントに保管します。戦利品を確保するには、指定された脱出ポイントまで移動し、無事に惑星を離れる必要があります。
  </Step>
</Steps>

<Callout type="warning" title="トランブラーの紛失">
トランブラーを失い、新しいものを構築するためのリソースがない場合は、トランブラーを所有しているクルーに参加するか、新しいキャラクターを作成する必要があります。軌道ステーションでのリソース管理は、惑星での生存と同じくらい重要です。
</Callout>
