<Steps>
  <Step num="1" title="トランプラーの初期化">
    出撃前に、トランプラー・エディターを使用して拠点を構成します。初心者の方は、少なくとも1つのストレージコンパートメントと基本的な装甲板を備えたバランスの良いレイアウトに焦点を当ててください。車両が破壊された場合に備え、すぐに再建できるよう設計図として保存しておきましょう。
  </Step>
  <Step num="2" title="モードの選択">
    最初の数回は **ボヤージュ（Voyage）モード** を選択してください。このモードは永続的なオープンワールド体験を提供し、リスクが低いため、迫り来る砂嵐の圧力を感じることなく、マップのレイアウトやリソースの場所を学ぶことができます。
  </Step>
  <Step num="3" title="探索と略奪">
    放棄された都市や難破船をターゲットにします。武器箱やリソースの山を探しましょう。トランプラーは主要な保管庫として機能するため、個人のインベントリからアイテムを移すために頻繁に車両へ戻りましょう。
  </Step>
  <Step num="4" title="防衛と脱出">
    地平線に煙や機械の動きがないか常に注意を払ってください。他のプレイヤーに遭遇した場合は、現在の戦利品の価値に基づいて戦うか逃げるかを判断します。脱出地点に到達し、軌道輸送機が到着するまでそこを防衛してください。
  </Step>
</Steps>

<Callout type="success" title="生存のヒント">
常に少なくとも2つの修理キットを携帯してください。各キットには3回の使用回数があり、脱出地点まで這って辿り着くか、砂丘で立ち往生するかの分かれ目になります。
</Callout>

<Section icon={Shield} title="トランプラーのビルドと設計図管理">

トランプラーは単なる乗り物ではありません。移動要塞であり、保管ユニットであり、リスポーン地点でもあります。モジュール式ビルドシステムにより自由なカスタマイズが可能ですが、成功しているプレイヤーは通常、クルーの人数や目的に応じて特定の「役割」に特化させています。

<CardGrid cols={3}>
  <Card title="ステルススカウト">
    - **小型設計**: 遠くから見つかりにくい。
    - **高速移動**: 迅速な脱出のために最適化されたエンジン。
    - **ソロ向け**: 素早い略奪と離脱のために設計。
  </Card>
  <Card title="重装要塞">
    - **最大装甲**: 砲撃に耐えるための積層プレート。
    - **搭載火力**: 360度防衛のための複数のタレット。
    - **要クルー**: すべてのシステムを管理するために4〜6人のプレイヤーが必要。
  </Card>
  <Card title="貨物運搬機">
    - **拡張ストレージ**: 大量の戦利品のための複数のコンパートメント。
    - **防衛迷路**: 侵入者を阻止するための複雑な内部レイアウト。
    - **ボヤージュ専門**: 長時間の資源収集に最適。
  </Card>
</CardGrid>

効果的な建造には、電力消費、重量、ユーティリティのバランスが必要です。以下の表を参考に、モジュールのアップグレードの優先順位を決めてください。

| モジュールタイプ | 優先度 | 主な機能 | 重要な要件 |
| :--- | :--- | :--- | :--- |
| **エンジン** | 高 | 移動速度と登坂能力を決定 | 十分な燃料/電力 |
| **リアクター** | 必須 | すべての電子機器、タレット、ドアに給電 | 内部スペース |
| **ストレージ** | 必須 | 死亡時に戦利品を失うのを防ぐ | 中央への配置 |
| **スポーン地点** | 高 | クルーがトランプラー内でリスポーン可能になる | クルールームへのアクセス |
| **装甲プレート** | 中 | 大砲や爆発によるダメージを吸収 | 重量許容量 |

<Callout type="tip" title="建造戦略">
最も価値のあるコンパートメント（ストレージとリアクター）はトランプラーの中央に配置してください。外側には「ダミー」のコンパートメントや空き部屋を作り、乗り込みの試みや爆発ダメージに対する緩衝材として機能させましょう。
</Callout>

</Section>

<Section icon={Swords} title="武器ティアリストと装備ランキング">

2026年のメタでは、戦闘はプレイヤー同士の銃撃戦と、トランプラー同士の艦隊戦のようなエンゲージメントに分かれています。乗り込みや探索には個人武器が重要ですが、砂漠での生存を左右するのはトランプラーに搭載されたタレットです。

| ティア | アイテム名 | カテゴリ | 評価の理由 |
| :--- | :--- | :--- | :--- |
| **S** | **修理キット** | ユーティリティ | 戦闘中にトランプラーの耐久度を維持するために不可欠。 |
| **S** | **80mm砲** | 搭載武器 | 敵車両に対して壊滅的な長距離ダメージを与える。 |
| **A** | **ショットガンキャノン** | 搭載武器 | 乗り込みを狙う敵に対するクラス最高の防衛手段。 |
| **A** | **スモークグレネード** | 投げ物 | 配置転換や脱出の際に重要な視覚的遮蔽を提供。 |
| **B** | **オートキャノン** | 搭載武器 | 信頼性の高い持続射撃が可能だが、オーバーヒートしやすい。 |
| **B** | **M1866/9 Einzel** | プレイヤー用 | 中距離探索においてバランスの取れた万能ライフル。 |
| **C** | **WOK爆弾** | 特殊武器 | 高ダメージだが極めて稀。ドレッドノートでのみ発見可能。 |

<Callout type="info" title="マッチメイキングの仕組み">
公平性を保つため、マッチメイキングはクルーの人数に基づいています。ソロプレイヤーは1サーバー最大15人に制限されますが、大規模クルー（4〜6人）は合計最大45人のプレイヤーがいるサーバーで競い合います。
</Callout>
