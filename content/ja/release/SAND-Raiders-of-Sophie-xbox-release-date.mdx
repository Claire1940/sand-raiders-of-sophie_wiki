<Section icon={Swords} title="ゲームモード：Voyage（ボヤージュ） vs Storm Dive（ストームダイブ）">

最初のトランプラーを出撃させる前に、利用可能なゲームモードの違いを理解しておくことが重要です。『SAND: Raiders of Sophie』では、異なるリスク許容度やプレイスタイルに合わせた、惑星ソフィーの2つの探索方法が用意されています。

| 特徴 | Voyageモード | Storm Diveモード |
| :--- | :--- | :--- |
| **スタイル** | 常設型オープンワールド | バトルロイヤル型脱出 |
| **リスクレベル** | 低〜中程度 | 極めて高い |
| **戦利品の品質** | 標準的なスカベンジ | ティアの高い装備 / 希少なアーティファクト |
| **タイマー** | 遠征の時間制限なし | 縮小する砂嵐による制限あり |
| **脱出** | プレイヤーがいつでも任意に開始可能 | 砂嵐が閉じる前に脱出する必要あり |
| **おすすめの対象** | 初心者および資源集め（ファーミング） | ベテランのクルーおよびPvP戦闘 |

**Voyageモード**では、キャラクターとトランプラーの長期的な成長に焦点を当てます。迫り来る安全地帯の縮小によるプレッシャーを受けることなく、新しいビルドを試したり、自動生成されるマップの構造を学習したりするのに最適な環境です。一方、**Storm Dive**は最も価値のある戦利品が眠る場所ですが、高速で移動できるトランプラーと、激しいPvP戦闘に耐えうるクルーの準備が必要です。

<Callout type="warning" title="戦利品紛失に関する警告">
どちらのモードでも、トランプラーを失うと重大な資源の損失につながる可能性があります。遠征が失敗した場合に備え、お気に入りの設計図（ブループリント）からすぐに再建できるよう、拠点に十分な素材を常に保管しておくようにしてください。
</Callout>

</Section>

<Section icon={Target} title="最初の遠征の始め方">

ソフィーでの旅を始めるには、単なる武器だけでなく、移動式の要塞が必要です。最初の出撃が砂丘での敗北に終わらないよう、以下の手順に従ってください。

<Steps>
  <Step num="1" title="入手とインストール">
    SteamまたはXboxストアでゲームを購入します。少なくとも11GBのストレージ空き容量があることを確認してください。PCプレイヤーでLinuxやSteam Deckを使用している場合は、**Proton BattlEye Runtime**がインストールされていることを確認してください。
  </Step>
  <Step num="2" title="トランプラーエディター">
    出発する前に、トランプラーエディターに入ります。初期資源を使って、移動用の**エンジン**、電力用の**リアクター**、戦利品用の**ストレージ**といった必須コンパートメントを配置します。これを**設計図（ブルーフプリント）**として保存してください。
  </Step>
  <Step num="3" title="リージョンの選択">
    インベントリとテックツリーは、特定のリージョンに関連付けられています。最大90人のプレイヤー制限により、大規模な戦闘時にはネットワークへの負荷が高まるため、遅延（レイテンシ）を最小限に抑えるために最も近いサーバーを選択してください。
  </Step>
  <Step num="4" title="Voyageモードへの出撃">
    最初の数時間はVoyageモードに専念しましょう。小さな廃墟や放棄された街路を探索し、基本的なコンポーネントや武器の弾薬を集めることに集中してください。
  </Step>
  <Step num="5" title="脱出ループ">
    マップ上で脱出地点（エキストラクションポイント）を確認します。トランプラーをそのゾーンに移動させ、脱出シークエンスを開始します。脱出に成功すると、戦利品は永続的なインベントリに保護され、移動基地をアップグレードできるようになります。
  </Step>
</Steps>

<Callout type="success" title="初心者へのヒント">
2026年6月中旬に開催された公式の**サーバースラム**イベントに参加した場合は、インベントリに限定の**M1866/9 \"Einzel\" ブリーチローダー用スキン**が届いているか確認してください。これは早期テスト参加者向けの永続的なコスメティック報酬です。
</Callout>

</Section>

<Section icon={Shield} title="トランプラーのビルドとカスタマイズ">

トランプラーは、あなたの移動手段であり、防壁であり、主要な倉庫でもあります。『SAND: Raiders of Sophie』では、所持する武器と同様に、基地のレイアウトが極めて重要です。

  <Card title="スカウトウォーカー">

  <Card title="ルートホーラー">
    - **特徴**: 最大限のストレージ
    - 拡張されたカーゴコンパートメント
    - 防御用の煙幕発生装置
    - Voyageモードでの資源回収に特化
  </Card>

トランプラーはモジュール式になっているため、入り組んだレイアウトでコンパートメントを配置することができます。これは単なる見た目の問題ではありません。複雑な内部レイアウトは、戦闘中にあなたの船に乗り込もうとする敵プレイヤーを混乱させる役割も果たします。

| モジュールタイプ | 主な機能 | ゲーム序盤の優先度 |
| :--- | :--- | :--- |
| **修理キット** | 破損したモジュールの修復 | **必須**（3回使用可能） |
| **装甲プレート** | 船体の耐久度を向上 | 高（優先的な投資計画） |
| **クルールーム** | プレイヤー収容量を増加 | 中（スクアッドの場合は高） |
| **車載タレット** | 自動防衛機能 | 中（消費資源が多い） |

<Callout type="info" title="設計図システム">
設計したデザインは必ず保存しておきましょう。トランプラーが破壊されても、必要な素材さえ集めれば、設計図システムを使ってまったく同じレイアウトを迅速に再構築できます。
</Callout>

</Section>

<Section icon={Sparkles} title="システム要件と技術的なセットアップ">

広大な砂漠と90人のプレイヤーが交錯する環境を動作させるには、ハードウェアが特定のベンチマークを満たしている必要があります。本作は、公平なPvPvE環境を維持するために、カーネルレベルのアンチチートシステムである**BattlEye**を採用しています。

| コンポーネント | 必要動作環境 | 推奨動作環境 |
| :--- | :--- | :--- |
| **OS** | Windows 10 (64ビット) | Windows 11 (64ビット) |
| **プロセッサー** | Intel i5 (第8世代) / Ryzen 5 (2000シリーズ) | Intel i5 (第13世代) / Ryzen 5 (7000シリーズ) |
| **メモリ** | 16 GB RAM | 32 GB RAM |
| **グラフィック** | GTX 1650 / RX 570 / Arc A380 | RTX 3080 / RX 6800 XT |
| **ストレージ** | 11 GB の空き容量 | 11 GB（SSD推奨） |

Xboxプレイヤー向けには、Series Xが4K解像度/60 FPSを目標としており、Series Sは安定した1440pでの動作を目指しています。すべてのバージョンでフルコントローラーサポートが標準対応しており、PlayStation 5版ではDualSenseの振動機能もサポートされます。

<Callout type="warning" title="アンチチートに関する注意">
『SAND: Raiders of Sophie』のプレイにはBattlEyeが必要です。PCで起動に関する問題が発生した場合は、バックグラウンドで競合するソフトウェアが実行されていないか確認してください。Linuxユーザーは、Steamライブラリのプロパティから「Proton BattlEye Runtime」を有効にする必要があります。
</Callout>
