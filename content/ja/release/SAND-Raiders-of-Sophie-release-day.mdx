<Section icon={Sparkles} title="ゲームプレイモードの理解">

**SAND Raiders of Sophie 発売日**において、プレイヤーには砂漠を体験する2つの主要な方法、**ボヤージュ**と**ストームダイブ**が紹介されます。リスクと報酬の比率が大きく異なるため、進行状況に合わせて適切なモードを選択することが不可欠です。

ボヤージュは長期的な持続性と基本の習得のために設計されており、ストームダイブはバトルロイヤルゲームに似た、脱出要素のある高強度のセッションベースの体験を提供します。

| 特徴 | ボヤージュモード | ストームダイブモード |
| :--- | :--- | :--- |
| **主な目的** | 探索と資源収集 | 高価値戦利品とPvP生存 |
| **リスクレベル** | 中程度 | 非常に高い |
| **制限時間** | なし（永続的） | 砂嵐による制限あり |
| **戦利品の質** | 標準 / コモン | レア / レジェンダリー |
| **プレイヤー上限** | クルーサイズにより異なる | 最大90人 |
| **推奨対象** | 初心者 / ビルドテスト | ベテラン / 競争的な分隊 |

<Steps>
  <Step num="1" title="モードを選択する">
    最初のトランブラーを建造している場合はボヤージュを選びましょう。戦闘準備の整った要塞とフルクルーがいる場合はストームダイブを選びます。
  </Step>
  <Step num="2" title="トランブラーを準備する">
    エディターに入り、エンジン、リアクター、ストレージを設置します。ストームダイブでは、装甲プレートと設置型タレットを優先してください。
  </Step>
  <Step num="3" title="ソフィーへ出撃する">
    プロシージャル生成されたセクターに着陸します。マップを使用して、廃墟となった都市や難破船などのPOI（重要地点）を特定します。
  </Step>
  <Step num="4" title="安全に脱出する">
    脱出ポイントを見つけます。ストームダイブでは、砂嵐がマップの中央に到達する前に脱出しなければなりません。
  </Step>
</Steps>

<Callout type="tip" title="ソロでの生存">
ソロでプレイする場合は、「ステルスビルド」のトランブラーを使用しましょう。これらは小型で高速、地平線上で見つかりにくいのが特徴です。エリアに敵対分隊がいないことが確実でない限り、ドレッドノートPOIは避けてください。
</Callout>
