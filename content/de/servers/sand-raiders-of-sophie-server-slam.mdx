<Section icon={Sparkles} title="Update #1: Patchnotizen und Verbesserungen">

Basierend auf dem Feedback aus dem **sand raiders of sophie server slam** veröffentlichten die Entwickler am 26. Juni 2026 das Update #1. Dieses Patch zielte speziell auf das „klobige“ Bewegungsgefühl und die visuelle Klarheit der Wüstenumgebung ab. Der Character Controller wurde überarbeitet, um die Häufigkeit zu verringern, mit der Spieler an Umgebungscollidern hängen bleiben, insbesondere während der Hochgeschwindigkeitsbewegung der Trampler.

Neben technischen Fixes führte das Update Onboarding-Benachrichtigungen ein. Diese sollen neue Teams durch die Grundlagen der Expedition führen und die steile Lernkurve reduzieren, die während der ersten Phasen des Server Slams beobachtet wurde.

<Section icon={Shield} title="Serverstabilität und technische Herausforderungen">

Trotz der Erfolge des **sand raiders of sophie server slam** bleiben technische Hürden ein Hauptanliegen der Community. Das Spiel war anhaltenden DDoS-Angriffen ausgesetzt, was die Implementierung neuer Schutzschichten erforderlich machte. Ein besonders schwerwiegendes Problem betrifft den „VOYAGE“-Modus, der derzeit unter einem Speicherabsturz leidet. Dieser Absturz lässt Spieler auf einem Wiederverbindungsbildschirm hängen und sperrt sie effektiv aus ihrer aktuellen Sitzung aus.

| Problemtyp | Symptom | Antwort der Entwickler |
| :--- | :--- | :--- |
| **Speicherabsturz** | Verbindungsabbrüche im VOYAGE-Modus | VOYAGE bis zum Fix meiden |
| **DDoS-Angriffe** | Serverweite Latenz und Ausfälle | Schutzmaßnahmen implementiert |
| **Extraktions-Schleife** | Verlust von Beute nach erfolgreicher Extraktion | Wird untersucht |
| **Solo-Matchmaking** | Einzelspieler werden gegen große Crews gematcht | Hohe Priorität |

<Callout type="warning" title="Kritische Warnung">
Wenn Sie während der Extraktion die Nachricht „Verbindung zum Server verloren“ erhalten, beenden Sie das Spiel nicht sofort. Versuchen Sie eine einmalige Wiederverbindung, aber beachten Sie, dass anhaltende Schleifen auf eine tote Serverinstanz aufgrund des Speicherlecks hindeuten können.
</Callout>

Die Entwickler verfolgen auch „schwitzige“ Gameplay-Muster, bei denen erfahrene Spieler mit Endgame-Tramplern die Startgebiete dominieren. Obwohl das Spiel von *Sea of Thieves* inspiriert ist, debattiert die Community aktiv über die Notwendigkeit eines dedizierten PvE-Modus oder eines besseren Matchmakings zum Schutz neuer Scavenger.

</Section>
