<Callout type="info" title="Kurzanleitung">
- Die **Systemanforderungen für SAND: Raiders of Sophie** unterscheiden sich erheblich zwischen den Mindest- und den empfohlenen Hardware-Konfigurationen.

<Section icon={Shield} title="Optimierung deines Systems">

Über die reine Hardware hinaus spielt die Konfiguration deiner Software-Umgebung eine entscheidende Rolle für die Performance. Da das Spiel das BattlEye-Anti-Cheat-System auf Kernel-Ebene verwendet, solltest du sicherstellen, dass dein System frei von störenden Hintergrundprozessen ist.

| Optimierungsaufgabe | Nutzen |
| :--- | :--- |
| **GPU-Treiber aktualisieren** | Verhindert Grafikfehler und verbessert das Frame-Pacing |
| **SSD-Installation** | Verkürzt die Ladezeiten für die prozedural generierte Welt |
| **Hintergrund-Apps bereinigen** | Gibt RAM für die komplexe Trampler-Physik-Engine frei |
| **DirectX 11 prüfen** | Stellt die Kompatibilität mit der Kern-Rendering-API des Spiels sicher |

<Callout type="warning" title="Informationen zum Anti-Cheat">
Dieses Spiel verwendet BattlEye, ein Anti-Cheat-System auf Kernel-Ebene. Falls du das Spiel deinstallierst, denke daran, dass du die BattlEye-Dateien möglicherweise manuell entfernen musst, um sie vollständig von deinem System zu löschen.
</Callout>

</Section>
