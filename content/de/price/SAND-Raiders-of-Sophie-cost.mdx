<Section icon={Cpu} title="Systemanforderungen: Die Kosten der Leistung">

Neben den monetären Kosten für **SAND: Raiders of Sophie** müssen Spieler auch die Hardware-„Kosten“ berücksichtigen, um diesen ehrgeizigen Extraction-Shooter auszuführen. Das Spiel bietet eine riesige, prozedural generierte Welt mit komplexen zerstörbaren Elementen und riesigen mechanischen Festungen, weshalb die CPU- und RAM-Leistung von entscheidender Bedeutung ist.

| Komponente | Mindestanforderungen | Empfohlene Anforderungen |
| :--- | :--- | :--- |
| **Betriebssystem** | Windows 10 64-Bit | **Windows 11 64-Bit** |
| **Prozessor** | Intel i5 (8. Gen.) / Ryzen 5 (2000) | **Intel i5 (13. Gen.) / Ryzen 5 (7000)** |
| **Arbeitsspeicher** | 16 GB RAM | **32 GB RAM** |
| **Grafikkarte** | GTX 1650 / RX 570 / Arc A380 | **RTX 3080 / RX 6800 XT** |
| **DirectX** | Version 11 | **Version 11** |
| **Speicherplatz** | 11 GB verfügbarer Speicherplatz | **11 GB verfügbarer Speicherplatz (SSD)** |

<Steps>
  <Step num="1" title="Speicherplatz und RAM prüfen">
    Stellen Sie sicher, dass Sie mindestens 11 GB freien Speicherplatz haben. Obwohl der Speicherbedarf gering ist, ist die **Empfehlung von 32 GB RAM** entscheidend, um die Stabilität bei Begegnungen mit 90 Spielern aufrechtzuerhalten.
  </Step>
  <Step num="2" title="Proton BattlEye installieren (Linux/Deck)">
    Wenn Sie auf Linux oder dem Steam Deck spielen, navigieren Sie zu Ihrer Steam-Bibliothek und installieren Sie die **Proton BattlEye Runtime**, bevor Sie das Spiel starten.
  </Step>
  <Step num="3" title="Region auswählen">
    Wählen Sie Ihre Serverregion sorgfältig aus. Beachten Sie, dass Ihr **Inventar- und Technologiebaum-Fortschritt** an die spezifische Region gebunden ist, die Sie beim Start auswählen.
  </Step>
</Steps>

<Callout type="success" title="Speicher-Tipp">
Die relativ geringe Installationsgröße (11 GB) macht dieses Spiel zu einem hervorragenden Kandidaten für die Installation auf schnelleren NVMe-SSDs, was die Ladezeiten beim Übergang von der Orbitalstation zur Planetenoberfläche erheblich verkürzt.
</Callout>
